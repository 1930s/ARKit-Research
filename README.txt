Augmented Reality Exploration with ARKit

An undergraduate research project exploring the capabilities of ARKit for the Virginia Tech Department of Computer Science. The website contains a literature review of augmented reality, tutorials on how to get started with ARKit, and a description of the semester project.

Getting Started
To run the site from the command line, `cd` to the project's root directory, start a simple web server with `python -m SimpleHTTPServer`, and point your browser to http://localhost:8000/literature-review-site.

Sample Projects
Visit the demo-apps folder to view the source code of the applications built in the tutorials on the website. Or, follow the tutorials and develop them yourself!

Semester Project
Point your browser to http://localhost:8000/literature-review-site/semester-project.html to view a short overview of the semester project. View the source demo-apps/VT-AR-Tour/
