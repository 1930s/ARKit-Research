Creating the project

This project requires Xcode 9.0 and above.

  1. Launch Xcode and select Create a new Xcode project from the Xcode welcome screen, or select File → New → Project...
  2. Choose iOS → Application → Augmented Reality App and click next.
  3. In the Choose options for your new project dialog, choose SceneKit as the Content Technology

Configuring Your Environment

Creating a new ARKit project generates a sample application, so some cleanup is required.

  1. In ViewController.swift, delete the sample code from viewDidLoad().
  2. Delete everything inside the art.scnassets folder.

Create a new AR Session using the technique learned in the ARKit Hello World tutorial.

Automatic Lighting

In order to make the models you display to the user more realistic, the lighting applied to the models must match the real world lighting as closely as possible.

In the models and physics tutorial, we used a SCNView property called autoenablesDefaultLighting. This property creates an omnidirectional light source and places it at the camera. This way, as you move around the scene, it looks as if you are shining a light on each model.

Disable the automatic lighting by changing your sceneView.autoenablesDefaultLighting line to false.
