Results

The semester project combines the skills learned from reviewing the sample apps and creating the tutorial apps.

VT AR Tour


VT AR Tour allows the user to walk around the Virginia Tech campus and discover the buildings around them. By looking through the device's camera, the user can see the names of every Virginia Tech building--just by looking at them. To see more information, the user can tap on the building for more information, or just walk closer.

The app pulls data from the Virginia Tech buildings web service, and is essentially an augmented-reality version of the Virginia Tech Buildings webpage.

Take a look at the source code for a more in-depth exploration of how it works.
